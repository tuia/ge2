CRUDficador versão 1.0

Este manual tem como função auxiliar na construção de novas telas dos softwares relacionados, com base nos padrões já estabelecidos, otimizando a usabilidade do produto final.

Seguir os padrões aqui apresentados é fundamental para manter o sistema conciso e usável. Este manual é dinâmico e deve ser atualizado pela equipe interna com os novos padrões estabelecidos.

Grid

Um sistema de grid serve para auxiliar na diagramação do conteúdo da página, posicionando e alinhando os elementos de forma proporcinal.

Para isso, utiliza-se linhas (.row) e colunas (.col) de tamanhos variados. Algumas regras são necessárias para seu melhor funcionamento:

  • Para as dimensões ficarem corretas, quando se utilizar o grid diretamente na página, ele deve estar dentro de uma div com a classe .container ou .container-fluid.
  • Utiliza-se as colunas (rows) para criar grupos horizontais de colunas.
  • O conteúdo deve estar dentro das colunas, e apenas estas devem ser as filhas diretas (elemento um nível abaixo no DOM) das linhas (rows).
  • Uma linha (row) possui até 12 colunas. A largura da coluna é definida pelo número no final da classe col-xs-, sendo .col-xs-1 o menor tamanho e .col-xs-12 o maior.

Botões

Para se utilizar um botão, basta adicionar a classe .btn e as classes da cor e tamanhos desejados ao elemento a, button, input [type="button"] ou input [type="submit"].

Cores

As cores são indicadas através das seguintes classes:

Exemplos

Código

<!-- Botão padrão -->
<button type="button" class="btn btn-default">Default</button>

<!-- Fornece peso visual extra e identifica a ação principal em um conjunto de botões -->
<button type="button" class="btn btn-primary">Primary</button>

<!-- Indica uma ação bem-sucedida ou positiva -->
<button type="button" class="btn btn-success">Success</button>

<!-- Botão contextual para mensagens de alerta informativo -->
<button type="button" class="btn btn-info">Info</button>

<!-- Indica que se deve tomar cuidado com esta ação -->
<button type="button" class="btn btn-warning">Warning</button>

<!-- Indica uma ação perigosa ou potencialmente negativa -->
<button type="button" class="btn btn-danger">Danger</button>

<!-- Retira a ênfase do botão, fazendo com que pareça um link e mantendo seu comportamento -->
<button type="button" class="btn btn-link">Link</button>

Dica! Você também pode utilizar as classes .active e .disabled para os estados ativo e inativado.

O dropdown select múltiplo substitui o elemento HTML select, e permite a seleção de vários itens por vez.

Este elemento não suporta outros elementos nas opções, como ícones. Utiliza o framework Bootstrap Select. Em sua documentação, você pode conferir outras opções para o elemento.

Exemplos

Código

    <option>van</option>

Tabelas

Existem vários tipos de tabelas que podem ser utilizados. Para todos os casos, a estrutura básica é a mesma, variando apenas as classes do elemento table

<table class="table">
    <thead>
      <tr>
        <th>...</th>
      </tr>
    </thead>
    <tbody>
      <tr>
        <td>...</td>
      </tr>
    </tbody>
  </table>

Dica! O elemento thead é utilizado para definir o cabelçalho da tabela, e para suas células, deve se utilizar a tag th.

O elemento tbody é utilizado para definir o corpo da tabela, e para suas células, deve se utilizar a tag td.

Existe também a tag tfoot, para definir o rodapé das tabelas. Seu uso é opcional.

Campo de texto com contador de caracteres

O exemplo a seguir limita o usuário a 30 caracteres e exibe um contador. É necessário colocar o número de caracteres máximo no atribulo maxLength, bem como ajustar os valores no código jQuery.

Exemplo

30

Código HTML

<div class="input-counter-container">
    <input type="text" class="form-control" placeholder="digite" maxlength="30">
    <div class="input-counter">30</div>
  </div>

Código jQuery

$('.input-counter-container .form-control').keyup( function() {
    var len = $(this).val().length;
      if (len >= 31) {
        $(this).val() = $(this).val().substring(0, 30);
      } else {
        $('.input-counter').text(30 - len);
        if (len > 24) { $('.input-counter').css('color', 'red'); }
        else { $('.input-counter').css('color', 'inherit'); }
      }
  });

O conteúdo da janela vem dentro de um iframe.

Exemplo

abrir janela de entidade

Código do botão

Para o botão chamar a janela, ele deve possuir a classe .show-window-btn e seu atributo href deve apontar para a id da janela que será exibida, nesse caso, #entity-window-example
