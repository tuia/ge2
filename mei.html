Manual de Estilos de Interação Axxiom versão 1.0

O Manual de Estilos de Interação tem como função auxiliar na contrução de novos compontentes dos softwares relacionados, com base nos padrões já estabelecidos, otimizando a usabilidade do produto final. Assim, o software se mantém consistente e os usuários sentem a diferença na produtividade.

Seguir os padrões aqui apresentados é fundamental para manter o sistema conciso e usável. Este manual é dinâmico e deve ser atualizado pela equipe interna com os novos padrões estabelecidos.

Sistema de Grid

Um sistema de grid serve para auxiliar na diagramação do conteúdo da página, posicionando e alinhando os elementos de forma proporcinal.

Para isso, utiliza-se linhas (.row) e colunas (.col) de tamanhos variados. Algumas regras são necessárias para seu melhor funcionamento:

  • Para as dimensões ficarem corretas, quando se utilizar o grid diretamente na página, ele deve estar dentro de uma div com a classe .container ou .container-fluid.
  • Utiliza-se as colunas (rows) para criar grupos horizontais de colunas.
  • O conteúdo deve estar dentro das colunas, e apenas estas devem ser as filhas diretas (elemento um nível abaixo no DOM) das linhas (rows).
  • Uma linha (row) possui até 12 colunas. A largura da coluna é definida pelo número no final da classe col-xs-, sendo .col-xs-1 o menor tamanho e .col-xs-12 o maior.
