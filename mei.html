Manual de Estilos de Interação Axxiom versão 1.0

Este manual tem como função auxiliar na construção de novas telas dos softwares relacionados, com base nos padrões já estabelecidos, otimizando a usabilidade do produto final.

Seguir os padrões aqui apresentados é fundamental para manter o sistema conciso e usável. Este manual é dinâmico e deve ser atualizado pela equipe interna com os novos padrões estabelecidos.

Grid

Um sistema de grid serve para auxiliar na diagramação do conteúdo da página, posicionando e alinhando os elementos de forma proporcinal.

Para isso, utiliza-se linhas (.row) e colunas (.col) de tamanhos variados. Algumas regras são necessárias para seu melhor funcionamento:

  • Para as dimensões ficarem corretas, quando se utilizar o grid diretamente na página, ele deve estar dentro de uma div com a classe .container ou .container-fluid.
  • Utiliza-se as colunas (rows) para criar grupos horizontais de colunas.
  • O conteúdo deve estar dentro das colunas, e apenas estas devem ser as filhas diretas (elemento um nível abaixo no DOM) das linhas (rows).
  • Uma linha (row) possui até 12 colunas. A largura da coluna é definida pelo número no final da classe col-xs-, sendo .col-xs-1 o menor tamanho e .col-xs-12 o maior.

Entidades

Elementos da rede, como alimentadores, dispositivos e equipamentos, são chamados no módulo de entidades.

Símbolos de entidade e status

Para melhorar sua identificação, elementos com a classe .entity-symbol possuem uma formatação diferenciada, para conter um ícone e um id ou informação relevante, bom como a possibilidade de ter esta informação copiada no clique. Eles também se diferenciam de acordo com o estado em que a entidade se encontra, variando de cor.

Exemplos

3564321 3564321 7894321 7894321 7894321 7894321
- .pendente .designado .acionado .em-execução .finalizado

Código

                

<a href="#" class="entity-symbol ex-execucao">
  <i class="fa fa-car"></i>
  <span>3564321</span>
</a>

Janelas de Entidades

Janelas de entidades (.entity-window) são elementos flutuantes arrastáveis pela tela, utilizadas para exibir ações e detalhes da entidade. Estas janelas são utilizadas no G-DIS e tem o objetivo de exibir o conteúdo sem retirar o mapa da evicência.

O conteúdo da janela vem dentro de um iframe.

Exemplo

abrir janela de entidade

Código do botão

Para o botão chamar a janela, ele deve possuir a classe .entity-icon e seu atributo href deve apontar para a id da janela que será exibida, nesse caso, #entity-window-example

                

<a href="#entity-window-example" class="btn entity-icon">abrir janela de entidade</a>

Dica! Para organizar seu código, você pode deixar as janelas de entidade sempre no final, já que elas ficam escondidas. Assim, você deixa todas elas juntas e pode focar no conteúdo principal da página.

Código da janela

                

<!--/JANELA DE ENTIDADE-->
<div class="entity-window" id="entity-window-example">
  <div class="entity-content">

    <!--/CABEÇALHO DA JANELA-->
    <div class="entity-window-header">
      <a href="#" class="bookmark-button hastooltip" title="adicionar aos favoritos"><i class="fa fa-bookmark-o"></i></a>
      <div class="pull-right">
        <a href="#" class="btn btn-xs btn-default hastooltip" data-title="destacar"><i class="fa fa-external-link"></i></a>
        <a href="#" class="btn btn-xs btn-default btn-close"><i class="fa fa-times"></i></a>
      </div>
    </div>

    <!--/CORPO DA JANELA-->
    <div class="entity-body">
      <!--/CONTEUDO-->
      <iframe src="conteudo-do-iframe.html" frameborder="0" scrolling="no"></iframe>
    </div>

  </div>
</div>
